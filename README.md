## Features

- Fully responsive design for mobile and desktop
- Ready to deploy on GitHub Pages

## Setup Instructions

### 1. Add Your Image

Place your image in the same folder as `index.html` and name it `image.jpg` (or update the filename in the HTML file). If you don't add an image, a placeholder heart image will be shown.

Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`

### 2. Customize the Message (Optional)

Edit lines 131-136 in `index.html` to personalize your romantic message.

### 3. Upload to GitHub

1. Create a new repository on GitHub
2. Upload both `index.html` and your image file
3. Go to Settings ‚Üí Pages
4. Under "Source", select "Deploy from a branch"
5. Select "main" branch and "/ (root)" folder
6. Click Save

Your website will be live at: `https://yourusername.github.io/repository-name`

## Customization Tips

### Change Colors

The main color theme uses red (#ff1744). To change it:
- Search for `#ff1744` in the HTML file
- Replace with your preferred color code

### Modify Text

- Line 103: Change "A" to your valentine's name
- Lines 131-136: Edit the romantic message
- Line 104: Modify button text if desired
